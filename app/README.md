# App - Scripto

Este é o diretório principal do seu aplicativo dentro do repositório Scripto.

## Estrutura
- `main.lua`: O ponto de entrada principal e motor da IA.
- `ia_logic.lua`: O módulo que contém a lógica de processamento de linguagem e automação.

## Funcionalidades de IA
1. **Chat Inteligente**: Capaz de reconhecer padrões de fala e responder ao usuário.
2. **Automação de Tarefas**: Pode ser configurado para executar rotinas como limpeza, backup e otimização de scripts.

## Como usar
Basta rodar o arquivo `main.lua` em um ambiente Lua. Você pode expandir o banco de dados de conhecimento no arquivo `ia_logic.lua`.
